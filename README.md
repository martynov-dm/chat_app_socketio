# Awesome Chat App

### Built with TypeScript, React/Redux and Socket.io

## Key Features

- Sign-Up/Sign-In with validation and api error popups
- Uploading user Images
- Groups and Servers
- Real-time communication
- Adding your own rooms
- Fully Responsive
- Light and Dark Themes
